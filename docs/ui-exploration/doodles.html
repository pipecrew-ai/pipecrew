✍️ Pipeline Crew — Doodles Edition

Hand-drawn action poses on notebook paper. Inspired by Open Doodles.
Each character is hand-coded SVG in Open Doodles style: thick black outlines (4px), wavy imperfect strokes, flat pastel fills, expressive action poses. Cards have squishy organic border radii and chunky drop shadows like stickers on a notebook page. Paper is ruled with faint blue lines.

For a production version, use real Open Doodles assets from opendoodles.com.
