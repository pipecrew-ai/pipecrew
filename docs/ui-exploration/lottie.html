🎬 Pipeline Crew — Lottie Edition

JSON animations rendered via lottie-web. Each character plays a programmatic animation.
lottie-web loaded from unpkg CDN. Each character renders a procedurally-generated Lottie JSON (a colored shape with a bounce animation) — this demonstrates the approach without requiring external animation files. In production, you'd export animations from After Effects and load them from character.json files via lottie.loadAnimation({ path: 'pip.json' }).

Sources for real character animations: LottieFiles.com (huge free library) or commissioned artwork.
